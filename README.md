# # 安装 Deno

```sh
curl -fsSL https://deno.land/x/install/install.sh | sh
```

```sh
Archive:  /Users/mockuai/.deno/bin/deno.zip
inflating: /Users/mockuai/.deno/bin/deno
Deno was installed successfully to /Users/mockuai/.deno/bin/deno
Manually add the directory to your $HOME/.zshrc (or similar)
  export DENO_INSTALL="/Users/mockuai/.deno"
  export PATH="$DENO_INSTALL/bin:$PATH"
Run '/Users/mockuai/.deno/bin/deno --help' to get started
```
